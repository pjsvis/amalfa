# Scripts: Verification

This directory contains scripts for debugging, integrity checks, and benchmarking.

## Contents

- **`verify_*.ts`:** Various integrity checks for graph, unification, and sync status.
- **`debug_*.ts`:** Tools to inspect specific nodes or edges.
- **`find_refs.ts`:** Find references to terms in the DB.
- **`benchmark_db.ts`:** Performance testing for database operations.
