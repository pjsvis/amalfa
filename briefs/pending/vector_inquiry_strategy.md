# VECTOR_INQUIRY_STRATEGY.md

**GOAL:** ENRICH THE GRAPH USING VECTOR INTELLIGENCE.

## OBJECTIVES
1.  **ORPHAN RESCUE:** FIND SEMANTIC PARENTS FOR DISCONNECTED NODES.
2.  **BRIDGE BUILDING:** CONNECT DISTINCT CLUSTERS (UX <-> DB).
3.  **DUPLICATE DETECTION:** MERGE REDUNDANT NODES.

## THE INQUIRY PROTOCOL

### QUERY 1: "WHO IS MY SIBLING?"
*   **TARGET:** ORPHAN NODES.
*   **METHOD:** VECTOR SIMILARITY > 0.85 TARGETING CONNECTED GRAPH.
*   **ACTION:** PROPOSE `RELATED_TO` EDGE.

### QUERY 2: "WHERE IS THE HIDDEN BRIDGE?"
*   **TARGET:** CROSS-COMMUNITY PAIRS (LOUVAIN A vs LOUVAIN B).
*   **METHOD:** SIMILARITY > 0.8.
*   **ACTION:** PROPOSE `INTERDISCIPLINARY` EDGE.

### QUERY 3: "ARE WE REPEATING OURSELVES?" (ECHO CHAMBER)
*   **TARGET:** GLOBAL.
*   **METHOD:** SIMILARITY > 0.95.
*   **ACTION:** MERGE OR LINK `VERSION_OF`.

## THE HYBRID PROTOCOL (GRAPH + VECTORS)

### QUERY 4: "THE WORMHOLE" (MISSING LINK)
*   **LOGIC:** GRAPH DISTANCE > 3 (Far) AND VECTOR SIMILARITY > 0.85 (Close).
*   **MEANING:** Two concepts are identical but structurally disconnected.
*   **ACTION:** PROPOSE `RELATED_TO` EDGE to bridge the gap.

### QUERY 5: "THE FALSE BRIDGE" (BAD LINK)
*   **LOGIC:** GRAPH EDGE EXISTS (Connected) AND VECTOR SIMILARITY < 0.4 (Divergent).
*   **MEANING:** These nodes are linked but talk about completely different things.
*   **ACTION:** FLAGGED FOR REVIEW (Potential Semantic Drift or Bad Tag).

### QUERY 6: "THE BRIDGE TO NOWHERE" (OUTLIER)
*   **LOGIC:** LOW DEGREE (Isolate) AND LOW VECTOR DENSITY (Far from Centroid).
*   **MEANING:** Irrelevant content or extreme niche.
*   **ACTION:** CONSIDER ARCHIVING.
