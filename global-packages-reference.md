## Update Commands

### Homebrew (system tools)
- Update a single package: `brew upgrade <package>`
- Update everything: `brew upgrade`

### Bun (JavaScript tooling)
- Update a single package: `bun update -g <package>`
- Update everything: `bun update -g`
