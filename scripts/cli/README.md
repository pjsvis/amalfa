# Scripts: CLI

This directory contains operational tools meant to be run manually by the user.

## Contents

- **`dev.ts`:** Starts the development environment.
- **`harvest.ts`:** CLI wrapper for the Harvester logic.
- **`promote.ts`:** CLI tool to promote "Candidate" terms to "Taxonomy" (modify files).
- **`normalize_docs.ts`:** CLI tool to batch-normalize markdown files.
